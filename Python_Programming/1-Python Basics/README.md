### 1. Syntax vs. Semantics

### 2. Variables & Dynamic Memory
- **Reference-Based Storage:** In Python, variables are labels pointing to objects in memory, not boxes holding values.
- **Dynamic Typing:** I explored how Python determines data types at runtime and the importance of type-checking to prevent production bugs.
